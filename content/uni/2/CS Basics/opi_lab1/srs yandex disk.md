## персональные заметки, мб использую их как нибудь
Сервис полностью бесплатен, но имеются платные возможности расширения доступного дискового пространства.
### Критика
История использования Яндекс Диска сохраняется навсегда. Пользователь не имеет возможности удалить историю своей активности.

### Основные платформы

- Web-версия. Находится внутри [Яндекс Почты](https://ru.wikipedia.org/wiki/%D0%AF%D0%BD%D0%B4%D0%B5%D0%BA%D1%81.%D0%9F%D0%BE%D1%87%D1%82%D0%B0 "Яндекс.Почта"), вкладка «Диск».
- Десктопный клиент для Windows и Linux
- Мобильная версия для [iOS](https://ru.wikipedia.org/wiki/IOS "IOS"), [Android](https://ru.wikipedia.org/wiki/Android "Android") и [Windows Phone](https://ru.wikipedia.org/wiki/Windows_Phone "Windows Phone").

### Поддерживаемые форматы файлов

Сервис Яндекс. Диск поддерживает онлайн-просмотр и проигрывание файлов следующих форматов:

- Документы — [DOC/DOCX](https://ru.wikipedia.org/wiki/Microsoft_Word "Microsoft Word"), [XSL/XSLX](https://ru.wikipedia.org/wiki/Microsoft_Excel "Microsoft Excel"), [PPT/PPTX](https://ru.wikipedia.org/wiki/Microsoft_PowerPoint "Microsoft PowerPoint"), [OpenDocument](https://ru.wikipedia.org/wiki/OpenDocument "OpenDocument"), [ODS](https://ru.wikipedia.org/wiki/OpenDocument "OpenDocument"), [ODP](https://ru.wikipedia.org/wiki/OpenDocument "OpenDocument"), [CSV](https://ru.wikipedia.org/wiki/CSV "CSV"), [RTF](https://ru.wikipedia.org/wiki/Rich_Text_Format "Rich Text Format"), [TXT](https://ru.wikipedia.org/wiki/%D0%A2%D0%B5%D0%BA%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D1%84%D0%B0%D0%B9%D0%BB "Текстовый файл"), [PDF](https://ru.wikipedia.org/wiki/Portable_Document_Format "Portable Document Format");
- Изображения — [JPEG](https://ru.wikipedia.org/wiki/JPEG "JPEG"), [PNG](https://ru.wikipedia.org/wiki/PNG "PNG"), [GIF](https://ru.wikipedia.org/wiki/GIF "GIF"), [BMP](https://ru.wikipedia.org/wiki/BMP "BMP"), [TIFF](https://ru.wikipedia.org/wiki/TIFF "TIFF"), [NEF](https://ru.wikipedia.org/wiki/Raw_\(%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%82_%D1%84%D0%BE%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%B9\) "Raw (формат фотографий)"), [EPS](https://ru.wikipedia.org/wiki/Encapsulated_PostScript "Encapsulated PostScript"), [PSD](https://ru.wikipedia.org/wiki/Photoshop_Document "Photoshop Document"), [CDR](https://ru.wikipedia.org/wiki/CDR_\(%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%82_%D1%84%D0%B0%D0%B9%D0%BB%D0%B0\) "CDR (формат файла)"), [AI](https://ru.wikipedia.org/wiki/Adobe_Illustrator_Artwork "Adobe Illustrator Artwork");
