# Перемещение программ
> Копировать программы средствами ОС — долго!

Модель аппаратного перемещения:
- Базовый регистр — указывает на начало программы
- Регистр границы — указывает на конец программы (иногда используется длина сегмента)
В программе закодирован относительный адрес ( в виде смещения ). Относительный адрес попадает в сумматор с базовым регистром → получаем абсолютный адрес. Далее идет сравнение с регистром границы для проверки, что адрес программы не вышел за границы (нельзя обратиться ниже регистра границы — вызовется прерывание)
